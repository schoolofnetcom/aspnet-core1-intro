# ASP.NET Core 1.0

Projeto destinado aos desenvolvedores ASP.NET, com o objetivo de apresentar os recursos do ASP.NET Core 1.0 para construção de aplicações web e cloud. Esta abordagem concede ao desenvolvedor uma visão de futuro do ASP.NET, que agora é open source, disponível no Git, para Windows, Linux e Mac. Além disso, o aluno se prepara para a tendência do mundo .NET, que é a evolução da plataforma, agora representada no ASP.NET Core, versão que foi reescrita para atender a necessidade de ser multiplataforma.
